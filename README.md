## Stack
- Platform: Node (v15.0.1)
- Web Framework: Express
- View engine: EJS
- DB: NoSQL
- ORM: Sequelize 
- Web App Host: Heroku
- Data host: PostgreSQL
- Coding standards: Prettier/ESlint
